📋 How to switch icons in the future

Open index.html in your GitHub repo (pencil icon to edit)
Find the line: <link rel="icon" href="icons/icon_frosted_glass.svg">
Change the filename to whichever icon you want, e.g. icons/icon_neon_m.svg
Also find <img src="icons/icon_frosted_glass.svg" in the header logo and update that too
Commit changes — done! ✅
Or just tell Claude "switch to Neon M icon" and it'll do it in seconds.
